## Emojis in Text

You can use emojis **inline** with text, like this: 🎈 for a balloon or 🍰 for a cake. Emojis can make your text more **expressive** and **fun**! 🎨✨

## Emojis in Code Blocks

```markdown
Here is a code block with an emoji: 🐍
```
